# FinVerse - Gamified Financial Intelligence Platform

> **This is a vibe coded app** - built entirely through AI-assisted development, turning creative vision into reality through natural language prompts and iterative refinement.

## What is FinVerse?

FinVerse is a next-gen gamified financial intelligence platform that transforms portfolio tracking into an immersive, game-like experience. Track investments, earn XP, complete financial quests, and compete on global leaderboards - all wrapped in a stunning dark neon aesthetic.

## Features

- **Particle Network Background** - Canvas-based floating particles with dynamic connecting lines
- **Glitch Text Animations** - Cyberpunk-style character decryption effects on headings
- **Live Stock Ticker** - Auto-scrolling marquee with real-time market symbols
- **KPI Command Center** - Animated counters for Portfolio Value, Returns, Win Rate, and XP
- **Portfolio Dashboard** - Animated donut chart with sparkline mini-charts for each holding
- **Gamification Engine** - XP system, levels, day streaks, and achievement badges
- **Quest Board** - Financial quests with progress tracking, difficulty tiers, and XP rewards
- **Global Leaderboard** - Competitive rankings with badges and streak tracking
- **Reduced Motion Support** - Full accessibility for users who prefer minimal animations

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, Shadcn UI, Framer Motion
- **Backend:** Node.js, Express
- **Animations:** Canvas API, Framer Motion, CSS Keyframes
- **Fonts:** Orbitron, Space Grotesk, JetBrains Mono

## Screenshots

The app features a dark neon theme with purple, cyan, and emerald accents throughout - designed to feel like a futuristic trading terminal meets RPG interface.

```bash
npm install

```

The app runs on `http://localhost:5000`.
